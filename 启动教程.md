# 步骤一

## 方法一：命令行方式

启动 MySQL 服务

```powershell
Get-Service -Name "*mysql*"
```
或者
```cmd
net start MySQL
```
如果 MySQL 服务未启动，会显示 "MySQL" 服务未运行。
可以试试加上mysql80，比如
```powershell
Get-Service -Name "*mysql80*"
```
或者
```cmd
net start MySQL80
```
只要看到Running或者大写的RUNNING，就说明 MySQL 服务启动成功。

--------

打开 MySQL 命令行客户端（或 Navicat/DBeaver 等 GUI 工具），执行以下 SQL：

```sql
-- 方法一：命令行方式
-- 打开 CMD 或 PowerShell，运行：
mysql -u root -p -e "source E:/2/news-system/backend/sql/newsdb.sql"
-- 输入你设置的 root 密码即可
```

**注意：**
-- 确保你复制到你本地的\news-system\backend\sql\newsdb.sql文件路径正确，不然启动失败

-- 文件
\news-system\backend\src\main\resources\application.yml
这里面的
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/newsdb?useUnicode=true&characterEncoding=utf-8&serverTimezone=Asia/Shanghai
    username: root
    password: 20051227  ← 根据实际情况修改
    driver-class-name: com.mysql.cj.jdbc.Driver

-- 这里的 url 是你本地的 newsdb 数据库连接字符串，根据实际情况修改。(基本上不要动就行了)

-- 这里的 username 是你本地的数据库 root 用户名（默认是 root），根据实际情况修改。(基本上不要动就行了)

-- 这里的 password 是你本地的数据库 root 密码，根据实际情况修改。(基本上不要动就行了)
```

# 步骤二

#### 后端启动

1. 用 Trae 打开项目文件夹 `news-system`
2. 左侧边栏找到 **Maven 面板**（Java Projects 下方）
3. 展开 **Lifecycle**
4. 双击 **`compile`**（首次需要编译）
5. 双击 **`spring-boot:run`**（如果没有这个选项，双击 `package` 打包后再运行）
6. 看到 `Started NewsApplication in x.xxx seconds` 即启动成功


-----

# 步骤三
#### 前端启动

```bash
cd frontend
npm install
```

3. 启动前端开发服务器：

```bash
npm run dev
```

4. 看到 `Local: http://localhost:5174/` 即启动成功
